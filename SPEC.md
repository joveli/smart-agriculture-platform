# 智慧农业平台 - 项目规划文档 (v1.4)

*完整规格文档见本文件附件，核心内容如下：*

## 项目概述

**项目名称**：智慧农业平台 (Smart Agriculture Platform)
**项目定位**：面向多租户的SaaS化智慧农业全流程管理平台，采用LLM智能体技术实现温室大棚智能控制，支持物联网设备集成、病虫害防治、农资管理、经济效益分析等核心功能。

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端Web | React + Ant Design |
| 前端小程序 | Uni-app (Vue) |
| 移动App | Flutter (V1.0后) |
| 后端API | Python FastAPI |
| LLM引擎 | MiniMax 2.5 |
| 物联网网关 | ThingsBoard / EMQX |
| 时序数据库 | TimescaleDB |
| 关系数据库 | PostgreSQL |
| 消息队列 | RabbitMQ |
| 缓存 | Redis |
| 对象存储 | MinIO / S3 |
| CI/CD | GitHub Actions |

## 开发阶段

| Phase | 内容 | 周期 |
|-------|------|------|
| Phase 1 | 基础架构搭建 | 4-5周 |
| Phase 2 | 核心业务开发 | 5-7周 |
| Phase 3 | 智能功能开发（LLM Agent） | 3-4周 |
| Phase 4 | 支付与合同 | 2-3周 |
| Phase 5 | 客户端开发 | 5-6周 |
| Phase 6 | 测试与部署 | 2-3周 |

**总体时间**：约25-28周（6-7个月）

## MVP 功能范围

**包含**：租户管理、温室监测、设备管理、基础告警、LLM问答、智能控制、合同管理、支付集成、Web后台、小程序

**不包含**：农场运营模块、病虫害图像识别、数据分析预测、Flutter App、开放平台API、政府对接、白标插件
